Hilfe anzeigen

Sie können einzelne Zeilen auswählen, indem Sie sie anklicken. Ausgewählte Zeilen werden in gelber Farbe dargestellt. Durch Anklicken der obersten Checkbox können Sie alle Zeilen gleichzeitig auswählen oder die Auswahl rückgängig machen.

Ausgewählte Zeilen können in einer Datei gespeichert werden, indem Sie auf den Link zum Speichern klicken.

Durch Klicken auf den Link zum Löschen rechts unterhalb der Tabelle werden die ausgewählten Zeilen aus der Tabelle entfernt. Vorsicht: Das Löschen kann nicht rückgängig gemacht werden.

Beim Klicken auf die Spaltenüberschriften werden die Zeilen der Tabelle auf- bzw. absteigend nach der jeweiligen Spalte sortiert.

Hinweis: Aus technischen Gründen funktioniert das Popup-Fenster zum Übersetzen auf dieser Seite nicht.
